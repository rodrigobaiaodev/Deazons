## 🚀 Sobre o Projeto

O **Deazons** é uma plataforma moderna e responsiva para entusiastas do audiovisual. Este projeto foi desenvolvido como parte dos meus estudos em **Engenharia de Software**, onde explorei a integração entre desenvolvimento de software tradicional e o uso de **Inteligência Artificial (IA)** para acelerar o processo de codificação e resolução de problemas técnicos.

Utilizamos a API oficial do **TMDB (The Movie Database)** para fornecer dados em tempo real sobre milhares de títulos, garantindo que as informações estejam sempre atualizadas para o usuário final.

## ✨ Funcionalidades Principais

- 🔍 **Busca Avançada**: Encontre filmes, séries e pessoas com filtros refinados.
- 🍿 **Últimos Trailers**: Uma seção dinâmica e interativa inspirada no TMDB para assistir aos trailers mais recentes.
- 📺 **Onde Assistir**: Integração com dados do **JustWatch** para mostrar em quais plataformas de streaming (Netflix, HBO Max, Prime Video, Disney+, etc.) o conteúdo está disponível no Brasil.
- 📱 **Interface Premium**: Design moderno com animações suaves, modo escuro nativo e foco total na experiência do usuário.
- 🇧🇷 **Localização Completa**: Todo o conteúdo é servido em Português do Brasil.
- ⚡ **Performance Otimizada**: Construído com **Vite** e **React** para carregamento quase instantâneo.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando o estado da arte do desenvolvimento web moderno:

- **React + TypeScript**: Para uma base de código robusta e tipada.
- **Tailwind CSS**: Para o estilização responsiva e moderna.
- **Shadcn UI**: Componentes de interface premium.
- **TMDB API**: Fonte de dados para filmes e séries.
- **Vite**: Ferramenta de build ultra-rápida.
- **Lucide React**: Biblioteca de ícones elegantes.
- **IA + Copilot**: Utilizei assistentes de IA avançados para arquitetar componentes complexos e otimizar rotinas de integração.

## 👨‍💻 O Estudante e a Engenharia de Software

Como estudante de Engenharia de Software, o objetivo deste projeto foi aplicar conceitos de:
- **Clean Code** e organização de diretórios.
- **Consumo de APIs Externas** de forma assíncrona.
- **Gerenciamento de Estado** em aplicações React.
- **Deploy Contínuo (CI/CD)** via Vercel.
- **Prompt Engineering**: Refinando a comunicação com IAs para gerar código de alta qualidade e interfaces premium.

## 📦 Como rodar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/rodrigobaiaodev/Deazons.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
